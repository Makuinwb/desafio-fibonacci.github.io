01 · Contexto

¿Cuál es el problema real?

En seguridad informática, los números primos son la base de los algoritmos de cifrado más importantes del mundo, como RSA. Para generar claves seguras se necesitan números primos grandes y difíciles de factorizar.

La serie de Fibonacci, por su parte, aparece constantemente en la naturaleza: en la espiral de los girasoles, la concha del nautilo, la reproducción de conejos. Sus números crecen de forma exponencial, igual que lo hacen los datos en sistemas digitales.

La pregunta interesante es: ¿cuántos números de Fibonacci son también números primos? Estos "Fibonacci primos" son especialmente raros y valiosos en teoría de números y criptografía experimental.

02 · Algoritmo

¿Cómo funciona?

A

Serie de Fibonacci

Cada número es la suma de los dos anteriores. Empezamos con 0 y 1, y el ciclo se repite tantas veces como términos pida el usuario. 

let a = 0, b = 1, c;
for (let i = 2; i <= n; i++) {
  c = a + b;
  a = b;
  b = c;
}
B

Verificación de Primo

Un número es primo si solo es divisible entre 1 y sí mismo. Contamos sus divisores: si hay exactamente 2, es primo. 

let contador = 0;
for (let i = 1; i <= num; i++) {
  if (num % i === 0) contador++;
}
// Si contador == 2 → es primo
C

Fibonacci Primos — La combinación

Generamos toda la secuencia de Fibonacci hasta N términos, y por cada número verificamos si también es primo. Los que pasen ambas condiciones son los "Fibonacci Primos" — los candidatos especiales en criptografía.

04 · Conclusión

¿Por qué importa esto?

🔐

Criptografía real

Los sistemas RSA que protegen tu banco, tu correo y tus contraseñas se basan en la dificultad de factorizar números primos grandes. Sin primos, no habría internet seguro.

🌿

Naturaleza y matemática

Fibonacci no es solo un ejercicio académico. Es el patrón con el que crecen las plantas, se organizan las semillas y se construyen espirales en toda la naturaleza.

🔍

Problema abierto

Hasta hoy, nadie ha podido demostrar si hay infinitos números que sean a la vez Fibonacci y primos. Es uno de los problemas abiertos más elegantes de las matemáticas.
